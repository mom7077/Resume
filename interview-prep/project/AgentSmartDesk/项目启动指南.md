## 技术栈

- **Java**: JDK 21
- **框架**: Spring Boot 3.2.0
- **数据库**: MongoDB 5.0+
- **向量存储**: Redis 6.0+ (需要 RediSearch 模块支持知识库RAG功能)
- **认证**: JWT (JSON Web Token)
- **AI集成**: Spring AI Alibaba (阿里云通义千问 DashScope)
  - AI Agent: 基于 Function Calling 机制自动选择工具
  - 会话记忆: SummaryBufferChatMemory（摘要缓冲记忆）
  - 知识库: RAG（检索增强生成）+ 向量存储
- **WebSocket**: Spring WebSocket
- **构建工具**: Maven 3.6+
- **PDF处理**: Apache PDFBox

## 前置要求

在开始之前，请确保您的电脑上已安装以下软件：

### 1. JDK 21 (必需)
- **版本要求**: JDK 21 或更高版本
- **下载地址**:
  - Oracle JDK: https://www.oracle.com/java/technologies/downloads/
  - OpenJDK: https://jdk.java.net/21/
- **验证安装**:
  ```bash
  java -version
  # 应该显示 java version "21.x.x" 或更高版本

  javac -version
  # 应该��示 javac 21.x.x 或更高版本
  ```

### 2. Maven (必需)
- **版本要求**: Maven 3.6 或更高版本
- **下载地址**: https://maven.apache.org/download.cgi
- **验证安装**:
  ```bash
  mvn -version
  # 应该显示 Apache Maven 3.6.x 或更高版本
  ```

### 3. MongoDB (必需)
- **版本要求**: MongoDB 5.0 或更高版本
- **下载地址**: https://www.mongodb.com/try/download/community
- **验证安装**:
  ```bash
  mongod --version
  # 应该显示 db version v5.x.x 或更高版本
  ```

### 4. Redis (必需)
- **版本要求**: Redis 6.0 或更高版本
- **模块要求**: RediSearch 模块（用于知识库向量存储和RAG功能）
- **推荐方式**: 使用 Redis Stack（已内置 RediSearch 模块）
  ```bash
  # Docker 方式（推荐）
  docker run -d --name redis-stack -p 6379:6379 redis/redis-stack:latest
  ```
- **其他安装方式**:
  - Windows: https://github.com/microsoftarchive/redis/releases
  - macOS: `brew install redis`
  - Linux: 通过包管理器安装
- **验证安装**:
  ```bash
  redis-server --version
  # 应该显示 Redis server v=6.x.x 或更高版本

  # 验证 RediSearch 模块（连接 Redis 后执行）
  redis-cli MODULE LIST
  # 应该包含 search 模块
  ```

## 配置文件说明

### 1. 检查配置文件

项目配置文件位于: `src/main/resources/application.yml`

### 2. 配置说明

打开 `application.yml` 文件，确认以下配置：

```yaml
spring:
  application:
    name: aiworkhelper

  # MongoDB 数据库配置
  data:
    mongodb:
      host: 127.0.0.1      # MongoDB 地址
      port: 27017          # MongoDB 端口
      database: aiworkhelper  # 数据库名称

    # Redis 配置（用于向量存储和缓存）
    redis:
      host: 127.0.0.1      # Redis 地址
      port: 6379           # Redis 端口
      database: 0          # Redis 数据库索引

  # Spring AI DashScope 配置
  ai:
    dashscope:
      api-key: your-api-key-here  # 请替换为您的 API Key
      base-url: https://dashscope.aliyuncs.com/compatible-mode/v1
      chat:
        options:
          model: qwen-max
          temperature: 0.7
      embedding:
        options:
          model: text-embedding-v3  # 用于知识库RAG的向量嵌入模型

# 服务器配置
server:
  port: 8888             # HTTP API 端口

# WebSocket 配置
websocket:
  server:
    port: 9000           # WebSocket 端口
    path: /ws            # WebSocket 路径

# JWT Token 配置
jwt:
  secret: kuangbiao.camps.aiworkhelper.jwt.secret.key.2024.secure.minimum.256bits
  expire: 8640000        # Token 过期时间（秒）= 100天

# AI Agent 记忆配置
ai:
  memory:
    # 最大Token限制（字符数估算）
    # 当对话超过此限制时，自动调用LLM生成摘要，保留关键信息
    max-token-limit: 2000

# 文件上传配置
upload:
  save-path: upload/     # 文件上传保存路径
  host: http://127.0.0.1:8888  # 服务器地址
```

**重要配置项说明**:

- **MongoDB**: 确保 MongoDB 服务已启动，默认端口 27017，用于存储业务数据
- **Redis**: 确保 Redis 服务已启动，默认端口 6379
  - 用于知识库的向量存储（需要 RediSearch 模块支持 RAG 功能）
  - 用于缓存和会话管理
- **DashScope API Key**:
  - 需要在 [阿里云 DashScope 控制台](https://dashscope.console.aliyun.com/) 申请
  - 免费额度：新用户有一定的免费调用额度
  - 替换配置文件中的 `api-key` 为您自己的密钥
  - 用于：AI 对话（qwen-max）、文本向量嵌入（text-embedding-v3）
- **AI Agent 记忆配置**:
  - `max-token-limit`: 当对话Token超过此限制时，自动调用LLM生成摘要
  - 摘要机制保留关键信息的同时控制Token消耗，避免上下文过长

## 启动项目

**先确保安装的mongodb和redis容器先启动**

### 方式一: 使用 Maven 启动 (推荐)

```bash
# 在项目根目录执行
mvn spring-boot:run
```

### 方式二: 使用 IDE 启动

1. 使用 IntelliJ IDEA 打开项目
2. 等待 Maven 依赖下载完成
3. 找到主类 `AIWorkHelperApplication.java`
4. 右键点击 -> `Run 'AIWorkHelperApplication'`

### 方式三: 打包后启动

```bash
# 1. 打包项目
mvn clean package -DskipTests

# 2. 运行 JAR 文件
java -jar target/aiworkhelper-0.0.1-SNAPSHOT.jar
```
